# vue-blog

博客后台管理：[vue-blog](https://github.com/linfeng-weng/vue-blog-admin)
博客后端：[vue-blog-server](https://github.com/linfeng-weng/vue-blog-server)

## 技术栈

- `Vue3` `Vue-Router` `Pinia` `Axios`
- 脚手架： `Vite`
- 组件库： `Element-Plus`

## 安装

1.安装依赖

```
    $ npm install
```

2.运行服务

```
    $ npm run dev
```
