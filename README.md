# vue-blog

后台管理：[vuets-blog-admin](https://github.com/linfeng-weng/vuets-blog-admin)
后端：[vue-blog-server](https://github.com/linfeng-weng/vue-blog-server)

## 技术栈

- `Vite` `Vue3` `Pinia` `Echarts` `WangEditor`
- 组件库： `Element-Plus`

## 安装

1.安装依赖

```
    $ npm install
```

2.运行服务

```
    $ npm run dev
```
